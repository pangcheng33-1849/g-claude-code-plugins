# 示例提示词

- 创建一个名为“Codex IM 验证群”的飞书群，并把庞铖拉进去
- 读取“测试claw”群最近 20 条消息
- 给“测试claw”群发一条文本消息，内容是“这是一条 Codex 验证消息”
- 给“测试claw”群发一条富文本长消息，包含标题、列表和链接
- 给“测试claw”群发一张卡片，里面用 Markdown 富文本展示上线说明
- 给“测试claw”群发一条正式同步消息，不要用 post，优先用 interactive 卡片
- 给“测试claw”群发一张分栏数据卡片，左边放增长指标，右边放风险提示
- 给“测试claw”群发一张带备注和按钮的通知卡片，整体风格简洁且层次清楚
- 对这条消息回复一句“已收到”
- 把这条消息编辑成“更新后的内容”
- 撤回这条测试消息
- 在这条消息上加一个活泼正向的 reaction，优先用 `THUMBSUP`、`THANKS` 或 `APPLAUSE`
- 读取这条消息当前的 reaction 列表
- 删除这条 reaction
- 在“测试claw”群发布一个话题，标题是“本周同步”
- 在这个话题下面回复“周一完成接口联调”
- 读取这个话题里的所有回复
- 编辑这个话题根消息
- 撤回这个话题
- 读取“测试claw”群的成员列表
- 把某个 open_id 对应的用户加进“测试claw”群
